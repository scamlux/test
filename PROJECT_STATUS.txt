МИКРОСЕРВИСЫ (5 сервисов)
  • api-gateway          → http://localhost:8000
  • order-service        → Управление заказами (Kafka)
  • inventory-service    → Управление инвентарём (Kafka)
  • payment-service      → Обработка платежей (Kafka)
  • query-service        → CQRS Read Model (Kafka) → http://localhost:8002

FRONTEND
  • web (React 18)       → http://localhost:3000

ИНФРАСТРУКТУРА
  • Docker Compose (локальная разработка)
  • Kubernetes Kind (продакшен-подобная среда)
  • PostgreSQL (заказы, инвентарь)
  • MongoDB (логи)
  • Kafka (эвент-стриминг)
  • Prometheus (метрики)
  • Loki (логи)
  • Grafana (визуализация)
  • OTEL Collector (трейсинг)
  • Tempo (сохранение трейсов)
  • Swagger UI (документация API)
  • ArgoCD (GitOps)
